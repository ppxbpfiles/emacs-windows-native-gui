# Emacs キーバインド チートシート

## CUA モード（Windows 互換）

| キー | 動作 |
|---|---|
| `C-c` | コピー |
| `C-x` | 切り取り |
| `C-v` | 貼り付け |
| `C-z` | アンドゥ |
| `C-y` | リドゥ |
| `C-s` | 上書き保存 |
| `C-a` | 全選択 |
| `C-o` | ファイルを開く（Windows ダイアログ） |
| `C-w` | バッファを閉じる |

---

## ファンクションキー

| キー | 動作 |
|---|---|
| `F2` | バッファ切り替え（consult-buffer）— centaur-tabs で非表示のタブも選択可能 |
| `F3` | インクリメンタル検索開始 / 次を検索（兼用・前方） |
| `S-F3` | インクリメンタル検索開始 / 前を検索（兼用・後方） |
| `F4` | アウトラインサイドバー開閉（imenu-list） |
| `F5` | howm 環境トグル（ON: メニュー / OFF: 全バッファ閉じる） |
| `F7` | 電卓（Calc）を開く |
| `Home` | ダッシュボードの表示 / 再描画（ダッシュボード上では閉じて元のバッファに戻る） |

---

## 検索・移動

| キー | 動作 |
|---|---|
| `C-f` | バッファ内ローマ字検索（Migemo） |
| `C-S-f` | ripgrep 全文検索（プロジェクト） |
| `C-S-g` | ファイル名検索（fd） |
| `C-r` | アウトライン検索（consult-outline） |
| `C-c l` | カーソル下の単語をポップアップ検索 |
| `C-c f` | エディター風フィルタ（マッチ行のみ表示・直接編集・一括保存） |
| `C-c F` | 複数ファイルの一括置換（wgrep + consult-ripgrep） |
| `C-c r w` | カーソル下の単語で ripgrep 検索 |
| `C-.` | カーソル位置でアクション（Embark） |

---

## フィルタ・grep編集（Occur / wgrep）

| キー | 動作 |
|---|---|
| `e` | (読み取り専用時) 編集モードに入る |
| `ESC` | (編集時) 編集モードを終了して読み取り専用に戻る |
| `C-c C-c` | 編集内容を確定して保存 (Occur: 元ファイルへ / wgrep: 全ファイルへ) |
| `C-c C-k` | (wgrep時) 編集内容を破棄して終了 |
| `q` | (読み取り専用時) バッファを閉じる |

---

## マルチカーソル（multiple-cursors）

| キー | 動作 |
|---|---|
| `C->` | 次の同じ単語にカーソル追加 |
| `C-<` | 前の同じ単語にカーソル追加 |
| `C-c m a` | バッファ内の全同一単語にカーソル追加 |
| `C-c m l` | 選択範囲の各行にカーソル追加 |
| `C-c m e` | 各行の行末にカーソル追加 |
| `RET` / `C-g` | マルチカーソル終了 |

---

## 矩形選択（rectangle-mark-mode）

| キー | 動作 |
|---|---|
| `Alt + ドラッグ` | マウスによる矩形選択を開始する |
| `C-RET` (Ctrl+Enter) | キーボードによる矩形選択を開始する |
| `ESC` | 矩形選択（または選択範囲）の解除 |

---

## カラーマーカー（symbol-overlay）

| キー | 動作 |
|---|---|
| `C-S-m` | マーカーをトグル（付ける / 外す） |
| `C-S-u` | 全マーカーを削除 |
| `C-S-n` | 次のマーカーへジャンプ |
| `C-S-p` | 前のマーカーへジャンプ |

---

## howm メモ

| キー | 動作 |
|---|---|
| `F5` | howm 環境トグル |
| `C-c #` | カーソル下の #タグ で howm 検索 |
| `C-c v` | 画像を貼り付け（org-download） |

---

## Obsidian 連携

| キー | 動作 |
|---|---|
| `C-c o f` | Obsidian ファイルを開く |
| `C-c o i` | Obsidian リンクを挿入 |
| `C-c o c` | リンク先ファイルを新規作成 |

---

## ダッシュボード（emacs-dashboard）

ダッシュボード（`*dashboard*`）バッファが表示されているときに、以下のキーで直接機能にアクセスできます。

| キー | 動作 |
|---|---|
| `Home` | ダッシュボードを閉じて直前のバッファに戻る |
| `o` | メインメニュー（Hydra ランチャー） |
| `e` | Everything で PC 内検索 |
| `s` | プロジェクト内全文検索（ripgrep） |
| `g` | プロジェクト内ファイル名検索（fd） |
| `f` | 最近使ったファイルを開く |
| `c` | cmd.exe（ConPTY ターミナル） |
| `p` | PowerShell（ConPTY ターミナル） |
| `L` | カレンダー（calfw） |
| `d` | 辞書（Lookup） |
| `w` | ウィンドウ操作サブメニュー |
| `F` | ファイル操作サブメニュー |

---

## M-o ランチャー（Hydra）

`M-o` を押してから続けてキーを入力します。

### メインランチャー（M-o）

| キー | 動作 |
|---|---|
| `e` | Everything で PC 内検索 |
| `s` | プロジェクト内全文検索（ripgrep） |
| `g` | ファイル名検索（fd・プロジェクト全体） |
| `G` | ここから以下のファイル名検索（fd） |
| `f` | 最近使ったファイル |
| `z` | Zoxide でファイルを開く |
| `r` | ローマ字検索（Migemo） |
| `l` | カーソル下の単語をポップアップ検索 |
| `n` | 新しいフレームを開く |
| `d` | 辞書（Lookup） |
| `P` | EPUB リーダー（nov） |
| `m` | Markdown サブメニュー → |
| `o` | Obsidian サブメニュー → |
| `w` | ウィンドウ操作サブメニュー → |
| `F` | ファイル操作サブメニュー → |
| `M` | カラーマーカーサブメニュー → |
| `L` | カレンダー（calfw） |
| `c` | cmd.exe（ConPTY） |
| `p` | PowerShell（ConPTY） |
| `C` | 電卓サブメニュー → |
| `T` | テキスト変換サブメニュー → |
| `q` | 閉じる |

### ウィンドウ操作（M-o w）

| キー | 動作 |
|---|---|
| `2` | 上下に分割 |
| `3` | 左右に分割 |
| `o` | 次のウィンドウへ移動 |
| `O` | 前のウィンドウへ移動 |
| `=` | 幅・高さをそろえる |
| `+` | 高さを広げる |
| `-` | 高さを縮める |
| `0` | このウィンドウを閉じる |
| `1` | 他のウィンドウをすべて閉じる |
| `k` | バッファを閉じる |
| `K` | バッファ＋ウィンドウを閉じる |

### ファイル操作（M-o F）

| キー | 動作 |
|---|---|
| `o` | ファイルを開く |
| `r` | 最近のファイル |
| `s` | 上書き保存 |
| `S` | 名前をつけて保存 |
| `k` | バッファを閉じる |
| `e` | 現在のファイルを外部アプリで開く |
| `E` | 任意のファイルを外部アプリで開く |
| `R` | 最近のファイルを外部アプリで開く |
| `m` | m3u/m3u8 プレイリストを検索して再生 |
| `a` | 現在のファイルを agy.exe に渡して cmd 外部窓で実行 |
| `A` | 現在のファイルを agy.exe に渡して PowerShell 外部窓で実行 |
| `x` | 任意の外部コマンドを cmd 外部窓で実行（%f=現在ファイルパス） |
| `X` | 任意の外部コマンドを PowerShell 外部窓で実行（%f=現在ファイルパス） |
| `d` | WinMerge で差分比較 |
| `c` | 文字コード指定で開き直す |
| `p` | Pandoc で別形式に変換 |
| `K` | EPUB を Kindle 形式に変換 |

### Markdown（M-o m）

| キー | 動作 |
|---|---|
| `1` | 見出し H1 挿入 |
| `2` | 見出し H2 挿入 |
| `b` | 太字（Bold）挿入 |
| `k` | 斜体（Italic）挿入 |
| `c` | コードブロック挿入 |
| `l` | リンク挿入 |
| `i` | 画像リンク挿入 |
| `t` | 目次（TOC）生成・更新 |
| `o` | 見出し検索ジャンプ（consult-outline） |
| `O` | アウトラインサイドバー開閉 |

### Obsidian（M-o o）

| キー | 動作 |
|---|---|
| `f` | ファイルを開く |
| `n` | 新規ノート作成 |
| `i` | リンクを挿入 |
| `c` | リンク先ファイルを作成 |
| `s` | Vault 内全文検索（ripgrep） |
| `r` | Vault 内ローマ字全文検索（Migemo） |
| `F` | Vault 内ファイル名検索（Migemo） |

### カラーマーカー（M-o M）

| キー | 動作 |
|---|---|
| `m` | マーカーをトグル |
| `u` | 全マーカーを削除 |
| `n` | 次のマーカーへ |
| `p` | 前のマーカーへ |
| `d` | 定義箇所へジャンプ |
| `c` | Casual メニュー |

### テキスト変換（M-o T）

| キー | 動作 |
|---|---|
| `s` | 選択行を昇順ソート |
| `S` | 選択行を降順ソート |
| `u` | 選択範囲の重複行を削除 |
| `z` | 選択範囲を半角 → 全角 |
| `Z` | 選択範囲を全角 → 半角 |
| `n` | 選択範囲に限定 (Narrow) |
| `w` | 限定を解除 (Widen) |

### 電卓（M-o C）

| キー | 動作 |
|---|---|
| `c` | Calc を開く |
| `m` | Casual メニューを開く |
| `a` | 代数モード（中置記法）に切り替え |
| `r` | RPN モード（スタック式）に切り替え |
| `0` | スタック全消去（AC） |
